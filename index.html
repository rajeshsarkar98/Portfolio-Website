Live Clock

A live clock with mode change effect is an enhanced version of a basic live clock that allows users to toggle between different time formats or visual themes (like light and dark modes). Using HTML, CSS, and JavaScript, the clock can display the current time in a 12-hour or 24-hour format. CSS provides the styling, while JavaScript is used to dynamically update the time and handle mode changes. A button or toggle allows users to switch between modes, applying different styles via CSS classes. This project teaches how to manipulate time, interact with user input, and create smooth transitions for a dynamic user experience.

Tic-Tac-Toe Game

Tic-Tac-Toe is a simple two-player game played on a 3x3 grid. One player uses "X" and the other uses "O." Players take turns marking empty spaces on the grid, aiming to get three of their marks in a row—horizontally, vertically, or diagonally. The game ends when one player achieves this or when the grid is full, resulting in a draw. Though easy to learn, Tic-Tac-Toe requires strategy to outsmart the opponent and anticipate their moves. It’s a popular pastime for children and adults alike, often played on paper, boards, or digital devices.
